
## Описание приложения
Было создано WinForm приложение предназначено для службы доставки, которое фильтрует заказы в зависимости от
количества обращений в конкретном районе города и времени обращения с и по.
Приложение позволяет пользователям в удобном табличном формате просмотреть все заказы и отфильтровать различными методами, а также добавить новую запись.

- net8.0-windows

- MSTest.TestAdapter (3.6.1)
- MSTest.TestFramework (3.6.1)
- NLog (5.3.4)
- NLog.Schema (5.3.4)
- NLog.Windows.Forms (5.2.3)
